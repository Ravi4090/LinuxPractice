Git Cheat Sheet

Serial No. Command Description Use Cases
1 git init Initialize a new Git repository Start a new project and create a new Git repository
2 git clone <repo> Clone a repository into a new directory Copy an existing repository to your local machine
3 git add <file> Add file(s) to the staging area Stage changes for the next commit
4 git status Show the status of changes as untracked, modified, or staged View the status of your changes
5 git commit -m "message" Commit staged changes with a descriptive message Record changes to the repository
6 git pull Fetch from and integrate with another repository Update the local repository with changes from the remote
7 git push Update remote repository with local changes Share committed changes with others by pushing to a remote repo
8 git branch List all branches in the repository View available branches in the repository
9 git branch <branch> Create a new branch Start working on a new feature or bugfix in a branch
10 git checkout <branch> Switch to a different branch Move between branches
11 git merge <branch> Merge changes from one branch into the current branch Combine changes from different branches
12 git log Display commit history View the commit history of the repository
13 git diff Show changes between commits or working directory View the differences between the working directory and the last commit
14 git reset <file> Unstage changes from the staging area Undo a previous git add
15 git remote -v List remote repositories and their URLs View configured remote repositories and their URLs
16 git fetch Download objects and refs from another repository Download changes from a remote repository without merging
17 git tag <tagname> Create a lightweight, annotated tag Mark a specific commit as a version
18 git pull origin <branch> Pull changes from a specific remote branch Update the local branch with changes from a specific remote branch
19 git remote add origin <repo> Add a new remote repository Set up a connection to a remote repository
20 git remote rm origin Remove the remote repository named 'origin' Disconnect from a remote repository
21 git remote rename <old> <new> Rename a remote repository Change the name of a remote repository
22 git remote show origin Show information about the remote repository 'origin' Display details about a remote repository
23 git push -u origin <branch> Set the remote branch as the upstream branch Establish a connection between the local and remote branches
24 git fetch --all Fetch changes from all remote repositories Retrieve changes from all remote repositories
25 git branch -a List all local and remote branches View all branches, including those from remote repositories
26 git checkout -b <new_branch> Create and switch to a new branch Combine branch creation and checkout into one command
27 git log --oneline --graph Compact, graphical representation of commit history Visualize the commit history with a concise graph
28 git cherry-pick <commit> Apply changes from a specific commit to the current branch Pick and apply specific changes to the current branch
29 git stash Temporarily save changes for later Stash changes to work on a different branch or address urgent issues
30 git stash apply Apply the last stash to the current branch Reapply the stashed changes to the working directory
31 git config --global user.name "Your Name" Set the author name globally Configure the global Git username
32 git config --global user.email "you@example.com" Set the author email globally Configure the global Git email
33 git log --grep="keyword" Show commits containing a specific keyword Filter commit history based on a keyword
34 git log -p Show full diff of each commit Display the complete changes for each commit
35 git branch -d <branch> Delete a local branch Remove a branch that is no longer needed
36 git push origin --delete <branch> Delete a remote branch Remove a branch from a remote repository
37 git rebase <branch> Reapply commits on top of another branch Modify commit history for a cleaner, linear history
38 git log --since="2 weeks ago" Show commits since a specific date View commits made within a specified time frame
39 git blame <file> Show who last modified each line of a file Identify the author and commit details for each line of a file
40 git log --stat Display commit statistics Show number of insertions and deletions in each commit
